User FAQs

Quick answers for common administrative tasks and issues

How do I reset a user's password as an user?

Navigate to Users → User Management → Select the user → Click 'Actions' → Choose 'Reset Password'. You can either generate a temporary password or send a password reset email to the user.

How do I add a new user to the system?

Go to User Management → Click 'Add New User' → Fill in details → Save.

How can I view system activity logs?

Navigate to Admin Panel → Logs → Filter and view activity records.

How do I change user roles and permissions?

Select user → Open 'Roles & Permissions' → Update roles → Save changes.

How do I deactivate or delete a user account?

Navigate to User Management → Select the user → Click 'Deactivate' or 'Delete'.

How do I configure email notifications for the system?

Go to Settings → Notifications → Configure SMTP and preferences.



User Best Practices and Guidelines

Essential guidelines and best practices for effective system administration

Security Best Practices

  • Enable two-factor authentication for all admin accounts
  • Regularly review user permissions and remove inactive accounts
  • Monitor login attempts ans set up alerts for suspicious activity
  • Keep system software and dependencies up to date
  • Implement strong password policies with regular rotation
  • Backup data regularly and test restoration procedures
  • Limit admin access to only neccessary personnel

System Maintanence Schedule

  • Daily Monitor system performance and error logs
  • Weekly:Review user activity reports and system health
  • Monthly:Update system software and security patches
  • Quarterly: Perform comprehensive security/audit
  • Quarterly:Review and archive old data
  • Anually:Conduct disaster recovery testing
  • As needed: Clean up unused accounts and performance
